# mysql-kafka-connect-study
가볍게 `mysql` source connector 학습하기

# 기술스택
* `spring boot`
* `mysql`
* `kafka`
* `kafka connect`
* `schema registry`
* `avro`

# 기능적 요구사항
* 게시판을 생성할 수 있다.
* 게시판에 댓글을 달 수 있다.
* `boardId`를 기준으로 게시판을 삭제한다.
* `userId`를 기준으로 전체 게시판을 삭제한다.
* 데이터 변경 이벤트를 구독해 데이터 생성 / 수정 / 삭제 로그를 출력한다.
